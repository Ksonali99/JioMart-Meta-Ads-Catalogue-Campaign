# 🛒 JioMart Meta Ads Catalogue Campaign Strategy

## 📌 Project Overview

This project focuses on designing a conversion-focused Meta Ads Catalogue Campaign for JioMart across Facebook and Instagram using dynamic product advertising, AI-driven audience targeting, and full-funnel performance marketing strategies.

The goal of this project is to improve:

* Purchase conversions
* Return on Ad Spend (ROAS)
* Customer acquisition
* Retargeting performance
* E-commerce sales growth

The campaign strategy is designed specifically for FMCG and e-commerce products using Meta’s catalogue advertising ecosystem.
